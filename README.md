# Smart Utility Hub

A responsive multi-tool web application built for the **Digital Heroes Trial Task**.
It combines three useful calculators in one clean and modern website:

* **GST Calculator**
* **EMI Calculator**
* **Age Calculator**

---

## GitHub Repository

**Repo URL:** `https://github.com/yuvarajaHM/digital-heroes-utility-hub`

---

## About the Project

**Smart Utility Hub** is a simple utility website designed to perform quick daily calculations in one place.

This project includes:

### GST Calculator

* Add GST to an amount
* Remove GST from an amount
* Supports common GST rates like **5%, 12%, 18%, 28%**

### EMI Calculator

* Calculate monthly EMI
* Calculate total interest
* Calculate total repayment amount

### Age Calculator

* Calculate exact age from date of birth
* Shows age in years, months, and days
